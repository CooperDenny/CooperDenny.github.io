Project Overview

The Victorian Property Prices Dashboard is an interactive Power BI project designed to offer comprehensive insights into the Victorian property market. This project highlights advanced data visualization techniques and dynamic interactions to facilitate the exploration of property price trends.

Features:

  • Dynamic map visualization of sold properties
  • Filterable slicers to analyze data by region, suburb, and property attributes
  • Interactive tables displaying property prices, locations, and key attributes

Data Source:

The data for this project was sourced from Kaggle, providing a detailed dataset of Victorian property sales.

Data Preparation:

For data preparation, I used Power BI to clean and preprocess the data, including:

  • Filtering out irrelevant entries and correcting errors
  • Transforming data to facilitate in-depth analysis and visualization
  • Creating calculated columns and measures for enhanced insights
